# login-react

login-react is a example of how implement login on your project React with Api-REST

## Installation
For Frontend
Clone repository
Run npm i command to install dependencies
Execute npm start command to run the project

for Backend

Clone repository
Run npm i command to install dependencies
Execute npm start command to run the project
